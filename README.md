# SagepayProtocol3

Ruby implementation of the SagePay payment gateway protocol v3.

## Installation

Add this line to your application's Gemfile:

    gem 'sagepay_protocol3'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sagepay_protocol3
